# Git コミットコマンド

リポジトリの最近のコミットスタイルを分析して、現在の変更を同じスタイルでコミットします。

## 指示

1. `git log --oneline -20` で最近のコミットメッセージのパターンを確認
2. `git status` で変更内容を確認
3. コミットスタイル(プレフィックス、言語、形式)に合わせてメッセージを生成
4. 適切なファイルをステージングしてコミット実行

## 例
```bash
# スタイル確認
git log --oneline -20

# 変更確認
git status

# ステージング & コミット
git add <files>
git commit -m "feat: 新機能追加"
```

## 注意

- 最近のコミットと同じ言語・形式を使う
- conventional commits形式なら従う
- 簡潔かつ明確に
