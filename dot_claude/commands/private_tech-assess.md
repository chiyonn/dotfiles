# Skill Assessment Update

今回のセッションを振り返り、ユーザーの技術力評価を更新する。

## 手順

1. **セッション振り返り**: 今回のやり取りで以下を特定
   - ユーザーが新しく学んだこと
   - つまずいたポイント・ギャップ
   - 既に知っていたこと（レベル確認）
   - 改善が見られた点

2. **既存ファイル確認**: `~/.claude/knowledge/user-tech-assessment.md` を読む

3. **更新判断**: 以下を検討
   - 新しい分野を追加すべきか
   - 既存分野のレベルを上げ下げすべきか
   - Learned セクションに今日の日付で追記
   - Gaps に新たな課題を追加、または解消した課題を削除
   - General Observations の更新が必要か

4. **ファイル更新**: 変更点をユーザーに説明しつつ、ファイルを編集

5. **Last updated を今日の日付に更新**

## 注意

- 客観的に評価する。お世辞は不要
- 小さな進歩も記録する（モチベーション維持）
- 繰り返し同じミスがあれば Gaps で強調
- ユーザーが自力で解決できた場合はレベルアップを検討
